## 💡 Example Prompt

```
Write Python code to find the maximum number in a list using a for loop.
```

---

## 🧪 Example Code

```python
def find_max(arr):
    if not arr:
        return None

    max_value = arr[0]
    for num in arr:
        if num > max_value:
            max_value = num
    return max_value


# Test
print(find_max([10, 24, 5, 32, 18]))
```
